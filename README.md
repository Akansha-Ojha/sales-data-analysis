# Sales Data Analysis Project

## Project Overview
This project analyzes a sales dataset using Python and Pandas to understand sales performance and identify best-selling products.

## Objectives
- Calculate total sales
- Find average and highest sales
- Identify best-selling product
- Generate basic report

## Tools Used
- Python
- Pandas
- VS Code
- GitHub

## Setup Instructions

1. Install Python (3.x)
2. Install Pandas:
   pip install pandas

3. Clone Repository:
   git clone https://github.com/Akansha-Ojha/sales-data-analysis.git

4. Run Program:
   python sales_analysis.py

## Technical Details
- Pandas used for data loading
- Data cleaning using fillna()
- GroupBy for product analysis
- Statistical calculations using mean(), sum(), max()
